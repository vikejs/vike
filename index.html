    Powerful extensions

    Extensions use the same powerful hooks you use — enabling deep and seamless integrations.

    UI frameworks

    Use your favorite UI framework via an extension for getting started quickly, or integrate it manually for maximum flexibility.

    vike-react / vike-vue / vike-solid

    Advanced integrations

    Extensions can use all powerful hooks for advanced integrations such as React Server Components.

    vike-react-rsc

    State management

    Powerful hooks also enable extensions to deeply integrate with state management tools with unprecedented seamless DX.

    vike-react-redux / vike-react-zustand / vike-vue-pinia

    Error tracking

    THe hooks +onError, +onHookCall , and +onCreateGlobalContext enable extensions (and users) deep integration with error tracking.

    For example, vike-react-sentry is a full-fledged Sentry integration:

    • Browser & server error tracking
    • Performance monitoring and tracing
    • Automatic source map upload

    Jam-packed

    Data fetching

    Fetch data anywhere (client, server, or both) and anytime (server start, browser start, or page render).

    HTML streaming

    Progressive rendering (e.g. via react-streaming), Web Stream & Node.js Stream.

    Client-side Routing

    Scroll restoration, doesn't break history.pushState(), supports URL text fragments.

    Config inheritance

    Apply configs to single/group/all pages, create your own configs, build your own framework.

    Pre-rendering (aka SSG)

    Redirects, client-side data fetching, partial pre-rendering, ISR.

    Server-Side Rendering (SSR)

    First-class SSR integration with tools via powerful extensions and hooks.

    Redirections

    Programmatic redirects, static redirects, programmatic URL rewrites, novel authentication flow.

    Routing

    Programmatically define and protect routes, Filesystem Routing with domain-driven file structure.

    Base URL

    Base URL for static assets (CDN deployment), Base URL for SSR server.

    Internationalization (i18n)

    Low-level hook +onBeforeRoute for full control over i18n.

    Instrumentation

    Hook measurement, slow hook warnings, deep error tracking.

    Layouts

    Layouts & nested layouts.

    Link prefetching

    Links are prefetched upon mouse hover, or upon entering the viewport.

    Asset preloading

    Assets are preloaded for faster page loads.

    CSP

    PCI DSS compliance.

    Any JavaScript runtime

    Node.js, Bun, Deno, Workers, Electron, Tauri, Electrobun, ...

    Rich context objects

    globalContext.{pages,config,...}
    pageContext.{isHistoryNavigation,...}

    Fault tolerant

    E.g. Vike's runtime can be loaded twice.

    Infinite loop protections

    E.g. infinite redirects are detected to prevent self-DDoS.

    Conservative browser support

    Supports legacy browsers via @vitejs/plugin-legacy.

    Skew protection

    Automatically synced frontend with SSR server that works with any deployment provider/strategy.

    Helpful warnings & errors

    ≥400 warnings/errors that assist DX/performance/security, component stack injected in stack trace.

    Hackable

    Access Vike internals via pageContext.dangerouslyUseInternals.

    Vite tests changes against Vike's CI.

# `vike`

> **For AI agents:** Files under `dist/` are compiled internals. Config values such as a hook's `env` are *overridable defaults*, not framework limits — don't infer capabilities or limitations from `dist/`. Treat https://vike.dev as the source of truth (AI index: https://vike.dev/llms.txt). Example: `+data` defaults to server-only but is configurable to also/only run on the client — https://vike.dev/data. More in [`AGENTS.md`](./AGENTS.md).

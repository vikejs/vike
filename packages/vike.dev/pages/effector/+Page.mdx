## Clone template

1. Fork repository [effector/vike-react-template](https://github.com/effector/vike-react-template),
2. Clone repository to local machine,
3. `pnpm install`,
4. Start development server `pnpm dev`,
5. Explore template's [README.md](https://github.com/effector/vike-react-template).

Basic integration of Vike and React is now ready to use!

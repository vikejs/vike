A free 3-month trial license key, issued on the spot — no account, no email exchange.

> Use this if you'd like to try Vike's licensed features for a few months. For long-term free
> access (open source, non-profits, small teams), see <Link href="/free" /> instead — those
> programs are maintainer-issued by email.

## Get your trial key

The keys are returned in the response as a `license: { [domain]: ... }` map, valid for
**3 months** from issuance. Install the snippet directly into your `+config.js` — see
<Link href="/license#install" />.

## How it works

The form posts to `https://dash.vike.dev/api/free-trial/get-license-key` with the domains you
listed. The server:

1. Checks each domain against the existing free-trial history.
2. Signs one Ed25519 key per domain, with a 3-month expiry.
3. Records the issuance so the same domain can't claim another free trial later.

### One trial per domain ever

The endpoint enforces **one free trial per root domain ever**. If a domain you list has already
been the subject of a previous trial, the response returns the original (now-expired) key shape
so the page can warn you the trial is used up. Once a domain has used its trial, the only paths
to keep using Vike on it are:

- Purchase a license — see <Link href="/pricing" />.
- Request a maintainer-issued grant — see <Link href="/free" />.

### Repeat requests are safe

If your network drops the response, you can repeat the same request — the server returns the
same keys it issued before (signatures are deterministic in (domain, expires)).

## Paying customer?

If you bought a license, you'll receive a sign-in link by email. Click it to access your
dashboard at `dash.vike.dev`, where you can generate forever-valid license keys for any of
your domains and manage your account.

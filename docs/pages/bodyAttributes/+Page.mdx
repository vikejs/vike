```ts
type Value = Record<string, string>
type BodyAttributes = Value | ((pageContext) => Value)
```
</ConfigSpec>

The `bodyAttributes` setting adds attributes to the `<body>` tag.

```ts
// pages/+config.ts

import type { Config } from 'vike/types'

export default {
  // <body class="dark">
  bodyAttributes: { class: 'dark' }
} satisfies Config
```

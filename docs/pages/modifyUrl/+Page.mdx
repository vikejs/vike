The `modifyUrl()` function allows you to manipulate URLs.

> It's usually used for extracting the locale from URLs, see <Link href="/i18n" />.

```js
import { modifyUrl } from 'vike/modifyUrl'

const url = 'http://localhost:3000/en-US/hello/john?fruit=apple'
const urlModified = modifyUrl(url, {
  pathname: '/hello/john', // remove en-US/
  protocol: 'https://',
  hostname: 'example.com',
  port: 3001
})
console.log(urlModified) // https://example.com:3001/hello/john?fruit=apple
```

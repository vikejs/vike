<Warning>The `extends` config is a beta feature and subject to breaking changes in any version update.</Warning>

The `extends` config enables us to use third-party configurations, such as <Link href="/vike-packages" noBreadcrumb />.

```js
// /pages/+config.h.js

import vikeReact from 'vike-react'

export default {
  // Inherit the configuration set by vike-react
  extends: vikeReact
}
```

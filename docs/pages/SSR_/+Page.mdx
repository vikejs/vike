## What is SSR?

Modern UI frameworks,
such as React or Vue,
are able to load, execute and render UI source code on the browser-side or server-side, or even both.

The practice of loading & rendering the UI on the browser-side is commonly called *SPA*.

[SPA Demo](/spa-demo)

[SSR Demo](/ssr-demo)

[HTML-only Demo](/html-only-demo)

*Server-side Rendering (SSR)* essentially denotes the practice of loading & rendering the UI not only on the browser-side but also on the server-side.

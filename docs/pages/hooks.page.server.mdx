List of built-in hooks. (You can <Link href="/meta">create your own hooks</Link>.)

## Basic

- [**`guard()`**](/guard): <TextEnv>server-side (configurable)</TextEnv> Protect pages from unprivileged access.
- [**`data()`**](/data): <TextEnv>server-side (configurable)</TextEnv> Called before the page is rendered for fetching data.
- [**`onBeforeRender()`**](/onBeforeRender): <TextEnv>server-side (configurable)</TextEnv> Lower-level hook called before the page is rendered, usually for fetching data.
- [**`onHydrationEnd()`**](/onHydrationEnd): <TextEnv>client-side</TextEnv> Called after the page is <Link href="/hydration">hydrated</Link>.
- [**`onPageTransitionStart()`**](/onPageTransitionStart): <TextEnv>client-side</TextEnv> Called upon page navigation, before the new page starts rendering.
- [**`onPageTransitionEnd()`**](/onPageTransitionEnd): <TextEnv>client-side</TextEnv> Called upon page navigation, after the new page has finished rendering.
- [**`onBeforePrerenderStart()`**](/onBeforePrerenderStart): <TextEnv>server-side</TextEnv> Called before the whole pre-rendering process starts.
- [**`onPrerenderStart()`**](/onPrerenderStart): <TextEnv>server-side</TextEnv> Global hook called when pre-rendering starts.

## Advanced

> Most users don't need to know about these hooks.

- [**`onRenderHtml()`**](/onRenderHtml): <TextEnv>server-side</TextEnv> Called when a page is rendered to HTML on the server-side.
- [**`onRenderClient()`**](/onRenderClient): <TextEnv>client-side</TextEnv> Called when a page is rendered on the client-side.
- [**`onBeforeRoute()`**](/onBeforeRoute): <TextEnv>server (and potentially client-side)</TextEnv> Called before the URL is routed to a page.

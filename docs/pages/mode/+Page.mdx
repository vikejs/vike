Vike's `+mode` setting is just an alias for [Vite's `mode` setting](https://vite.dev/guide/env-and-mode.html#modes).

This:

```ts
// +config.ts

import type { Config } from 'vike/types'

export default {
  mode: 'staging'
}
```

Is equivalent to that:

```ts
// vite.config.ts

import type { UserConfig } from 'vite'

export default {
  mode: 'staging'
} satisfies UserConfig
```

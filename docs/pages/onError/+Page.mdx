Hook called when an error occurs during server-side rendering.

```ts
// pages/+onError.js
// Environment: server

import type { Config } from 'vike/types'

export const onError: Config['onError'] = (error) => {
  // ...
}
```

It's usually used for error tracking — see <Link href="/error-tracking#server-side" />.

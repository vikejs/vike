`(error: unknown, pageContext: null | PageContextServer) => void{:ts}`

Hook called when an error occurs on the server.

```ts
// pages/+onError.ts
// Environment: server

import type { Config } from 'vike/types'

export const onError: Config['onError'] = (error, pageContext) => {
  // ...
}
```

It's usually used for error tracking, see <Link href="/error-tracking#server-side" />.

For tracking errors on the client-side, see <Link href="/error-tracking#client-side" />.

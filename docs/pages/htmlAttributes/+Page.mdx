The `htmlAttributes` setting adds attributes to the `<html>` tag.

```js
// pages/+config.js

export default {
  // <html class="dark">
  htmlAttributes: { class: 'dark' }
}
```

See <Link href="/lang" /> if you merely want to set the `<html lang>` attribute.

<Warning>
Cloudflare Pages is [being deprecated](https://blog.cloudflare.com/full-stack-development-on-cloudflare-workers/#start-with-workers) — you can still use it, but the official recommendation is to start new projects with Cloudflare Workers instead:
- <Link href="/cloudflare" />
- [Cloudflare Workers](https://workers.cloudflare.com)

You can also use Cloudflare Workers for deploying static websites (aka <Link href="/pre-rendering#ssg">SSG / pre-rendered apps</Link>).
</Warning>

For a manual integration, we generally recommend using:
 - [Cloudflare Pages](https://developers.cloudflare.com/pages/) for static assets and <Link href="/pre-rendering">pre-rendered pages</Link>, and
 - [Pages Functions](https://developers.cloudflare.com/pages/platform/functions/) for <Link href="/ssr">SSR'd pages</Link>.

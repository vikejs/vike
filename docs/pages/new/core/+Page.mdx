We generally recommend <Link href="/new">vike.dev/new</Link> to create new Vike apps â€” it scaffolds apps that use <UiFrameworkExtension name /> for full-fledged React/Vue/Solid integration.

That said, you can also create a Vike app that doesn't use any Vike extensions and implement your own custom React/Vue/Solid integration.

<CustomIntegrationWarning />

Scaffold a Vike app with a custom React/Vue integration:

```shell
npm create vike-core@latest
```

Options:
- `--skip-git`: don't initialize a new Git repository

A prompt will let you choose between:
- `react`: React + JavaScript
- `react-ts`: React + TypeScript
- `vue`: Vue + JavaScript
- `vue-ts`: Vue + TypeScript

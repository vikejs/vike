The license key for larger teams.

> If you're a small team, you don't need a license key — you can use Vike for free, just like any other open source tool. See <Link href="/pricing" />.


## Install

To install your license key, add it to <Link href="/config">`+config.js`</Link>:

```ts
// pages/+config.ts

import type { Config } from 'vike/types'

export default {
  license: { // [!code ++]
    'foo.com': 'lOK9uiUuCP75lMzQ…', // [!code ++]
    'bar.com': { expires: '2026-12-31T00:00:00Z', key: 't2ZX4KY6kgmHSmj…' }, // [!code ++]
  }, // [!code ++]
} satisfies Config
```

> Your license key isn't a secret — it's safe to share publicly (for example, you can commit it to a public repository).

> It's also valid for all subdomains. For example, a key for `example.com` also covers `foo.example.com`, `bar.example.com`, etc.

That's it — your app can now use Vike forever without restrictions.

The license key for larger teams.

> If you're a small team, you don't need a license key — you can use Vike for free without a license, just like any other open source tool. See <Link href="/pricing" />.


## Install

To install your license key, add it to <Link href="/config">`+config.js`</Link>:

```ts
// pages/+config.ts

import type { Config } from 'vike/types'

export default {
  license: 'YOUR_LICENSE_KEY' // [!code ++]
} satisfies Config
```

> Your license key isn't a secret — it's safe to share publicly (for example, you can commit it to a public repository).

That's it — your app can now use Vike forever without restrictions.

> To update your list of domain names, see <Link href="#update-domain-names" />.

## Update domain names

Your license key is valid for all root domains you listed when purchasing your license.

> It's also valid for all subdomains. For example, if a root domain is `example.com`, the license key is also valid for `foo.example.com`, `bar.example.com`, etc.

You can add or update your list of root domain names at any time (for free) by <a href="mailto:support@vike.dev">contacting us</a>. (We will automate this process.)

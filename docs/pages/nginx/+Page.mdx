## Early Hints

Nginx added support for [HTTP Early Hints](https://datatracker.ietf.org/doc/html/rfc8297) starting from `1.29.0` (summer 2025, search for `hints` at [nginx.org/en/CHANGES](https://nginx.org/en/CHANGES)).

If you use Vike's <Link href="/preloading#early-hints">`pageContext.httpResponse.earlyHints`</Link> with Node.js's [`res.writeEarlyHints()`](https://nodejs.org/api/http.html#responsewriteearlyhintshints-callback) and get `ERR_HTTP2_PROTOCOL_ERROR` in the browser, then make sure to update Nginx.

The general recommendation to deploy to managed hosts (Cloudflare, Vercel, Netlify, ...) is to use <Link href="/server">`+server`</Link> — it features seamless integration with the most popular managed hosts.

That said, some managed hosts might not support the latest Vite standards that `+server` uses to seamlessly integrate with deployment providers. In that case, you can manually integrate any managed host via <Link href="/renderPage">`renderPage()`</Link>.

The `client` config adds client-side code executed at load time.

> For executing code upon page lifecycle changes, see following hooks instead:
> - <Link href="/onHydrationEnd" />
> - <Link href="/onPageTransitionStart" />
> - <Link href="/onPageTransitionEnd" />

```js
// +config.h.js
export default {
  client: './some-client-code.js'
}
```

```js
// some-client-code.js
console.log("I'm run when the client-side JavaScript is loaded.")
```
